## 2026 ALGORITHM RULES — APPLY TO ALL CHANNELS

### Hard Rules
1. NEVER open with "Hey guys welcome back" — instant measurable drop-off
2. Script content must match title/description exactly — YouTube AI reads captions; mismatch kills suggested traffic
3. Never pad content to hit 8 minutes — retention loss > mid-roll gain
4. Never put the only CTA at the end — only 16% reach the final 10%
5. Every upload shown to strangers first — subscriber count irrelevant at upload time
6. AB test every video — 3 title variants, let YouTube pick winner to learn audience trigger

### Hook Structure (First 30 Seconds)
```
0:00–0:05  GRAB    — bold claim or unexpected moment (single most important line)
0:05–0:15  PROMISE — exact outcome viewer gets from watching
0:15–0:30  STAKES  — why it matters, what they lose by leaving
```
**Target:** 70%+ retention at 30 seconds

### Retention Engineering
- Pattern interrupt every 60–90s: [CAMERA CHANGE] [B-ROLL CUE] [GRAPHIC] [SOUND EFFECT] [UNEXPECTED STAT]
- Soft CTA at ~1 min — 60% of viewers still watching
- Open loop mid-video — suspension bridge = 68% higher completion
- Retention re-hook at ~60% mark — second hook moment
- Hard CTA in outro — maintain energy to last second, no wind-down

### Retention Benchmarks
| Metric | Target | Notes |
|--------|--------|-------|
| Retention at 30s | 70%+ | Solid hook |
| Retention at 30s | 80%+ | Exceptional |
| AVD | 70%+ | Viral zone — priority push |
| AVD | 50–60% | Solid |
| 8 min mark | Unlock | ~50% more revenue from mid-roll |
| First 5s interrupt | +23% | Higher retention |
| Open loops | +68% | Completion rate |
| Every 30s interrupt | +43% | Completion rate |
| 15%+ above channel avg | 2.3x | Algorithm push |

### AB Test Framework — Every Video
| Variant | Trigger | Pattern |
|---------|---------|---------|
| Curiosity | Unknown/secret/unsolved | "The Heist Nobody Can Explain" |
| Fear/Loss | Gone/destroyed/stolen/too late | "The $500M That Vanished Before Anyone Noticed" |
| Result/Number | Specific $, time, count | "2 Men Stole $500M in 81 Minutes" |

```
  Grab (0–5s): single most important line

  [PATTERN INTERRUPT] → content → [B-ROLL CUE] → micro-summary → forward hook

## SOFT CTA (~1 min mark)

## OPEN LOOP (mid-video): tease something for later

## RETENTION RE-HOOK (~60% mark)

## OUTRO: hard CTA — energy to last second
```

## AUTOMATED RESEARCH PIPELINE (Oracle Cron)

Runs every morning at 6am — feeds new ideas into Grimm Archives queue automatically:

```python
# ~/research-scheduler.py
import asyncio, json
from crawl4ai import AsyncWebCrawler
from datetime import datetime

TOPICS = [
    "List of art heists",
    "Lost treasure",
    "Financial scandal",
    "Hidden wealth history",
    "Unsolved theft",
    "Missing gold",
    "Billionaire secrets"
]

async def daily_research():
    results = []
    async with AsyncWebCrawler() as crawler:
        for topic in TOPICS:
            url = f"https://en.wikipedia.org/wiki/{topic.replace(' ','_')}"
            result = await crawler.arun(url)
            results.append({"topic": topic, "content": result.markdown[:2000], "scraped_at": datetime.now().isoformat()})
    with open(f"research-{datetime.now().strftime('%Y%m%d')}.json", "w") as f:
        json.dump(results, f, indent=2)
    print(f"Scraped {len(results)} topics")

asyncio.run(daily_research())

# Cron: 0 6 * * * python3 ~/research-scheduler.py >> ~/research.log 2>&1
```
